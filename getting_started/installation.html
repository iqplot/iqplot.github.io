Installation

You can install iqplot using pip. You will need the dependencies below installed.

$ pip install --upgrade iqplot

Dependencies

iqplot has the following dependencies.

  • Python 3.6 or newer

  • NumPy

  • Pandas

  • xarray

  • Bokeh

  • colorcet

Numba is suggested but not required.
